## SQL Sales Analysis Project

This project demonstrates SQL-style data analysis using Python, SQLite, and pandas.

### Tools Used
- Python
- SQLite
- pandas
- matplotlib

### What This Project Does
- Loads a dataset into a SQLite database
- Performs SQL queries to analyze sales and tipping behavior
- Aggregates revenue and tip statistics by day
- Identifies high-value transactions
- Visualizes revenue trends with a bar chart

### How to Run

Install dependencies:

python -m pip install -r requirements.txt

Run the script:

python sql_sales_analysis.py
